# Screenshots and Assets

This folder contains images and assets for the repository documentation.

## Needed Screenshots:

1. **example_qr.png** - Example of a generated QR code with logo
2. **demo.gif** - (Optional) Animated demo of the application in use
3. **screenshot_terminal.png** - Terminal showing the interactive menu

## How to Add Screenshots:

1. Run the application and generate a QR code
2. Take a screenshot of the generated QR code
3. Save it as `screenshots/example_qr.png`
4. Update README.md to include the image:

```markdown
## Example Output

![Example QR Code](screenshots/example_qr.png)
```

## Tips:

- Keep images under 1MB when possible
- Use PNG format for screenshots
- Use GIF for animations (keep under 5MB)
- Optimize images before committing
